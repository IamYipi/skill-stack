# skill-stack
Repositorio de skills reutilizables para productividad en agentes de IA. Incluye dos bloques separados:  Claude: skills modulares con instrucciones y recursos. Codex: skills estructuradas con contrato SKILL.md. Centraliza procedimientos para convertir agentes en herramientas especializadas.

Este repositorio está bajo custodia de José Antonio y Javier Garcia.
Son los únicos autores de estas skills.
El uso de las skills está bajo responsabilidad de cada individuo.

## Auto-installers

- Windows Codex PowerShell: `auto-installers/win/codex/install-codex-skills.ps1`
- Windows Codex doble clic: `auto-installers/win/codex/install-codex-skills.bat`
- Instala todas las skills bajo `codex/` en la carpeta local del usuario actual: `C:\Users\<usuario>\.codex\skills`
