## Operational prompt

On every non-trivial task, first determine whether any available skill is relevant and use it when applicable. If the task requires retrieving prior context, preferences, reusable solutions, writing conventions, or previously discovered steps, consult the archive skill before re-solving anything.

After completing the work, evaluate whether any newly learned information should be stored in archive. If the solution is confirmed to work, tests pass, and the issue is resolved, prepare a git commit only when working inside a git repository and following the archive conventions. Before committing, ask whether the task is work-related. If it is, find the corresponding Jira issue for the project and link the commit to it when such an issue exists.
